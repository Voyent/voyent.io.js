(static) sendPushEvent(params)

Push notification to a push group. This will result in an Ajax Push (and associated callback) to any web pages that have added a push listener to the specified group. If Cloud Push options are provided (params.subject and params.detail) a Cloud Push will be dispatched as a home screen notification to any devices unable to recieve the Ajax Push via the web page.
